# jHECHMS

HEC-HMS hydrological model extracted from [SYMFLUENCE](https://github.com/DarriEy/SYMFLUENCE).

Dual-backend (JAX + NumPy) implementation of core HEC-HMS algorithms:
ATI snow model, SCS-CN continuous loss, Clark unit hydrograph, and linear reservoir baseflow.

## Installation

```bash
pip install jhechms
```
